Frequently Asked Questions

What is the difference between Demo and Pro?
The Demo plan is free and designed for testing and prototyping with limited API calls. Pro plans (Analyst and Enterprise) offer significantly higher rate limits, access to premium endpoints including NFT data, WebSocket support, historical OHLCV data, and priority support.
How do I get an API Key?
Simply create a free CoinGecko account, navigate to the Developer Dashboard, and generate your API key. You'll be making your first API call within minutes — no credit card required for the Demo plan.
Does CoinGecko offer WebSocket support?
Yes. WebSocket support is available on Pro and Enterprise plans, enabling real-time streaming data for price updates, trade data, and order book changes without constant polling.
What are the rate limits for the Free plan?
The free Demo plan includes 10,000 API calls per month with a rate limit of 10-30 calls per minute. This is sufficient for testing, prototyping, and small personal projects.
How often is data refreshed?
Market data is refreshed every 30-60 seconds depending on the endpoint and your plan tier. Enterprise customers can access near real-time data with sub-30-second refresh intervals via WebSocket.
Can I access NFT floor prices?
Yes. NFT floor prices, market caps, volume data, and collection metadata are available via dedicated NFT endpoints. This data covers major marketplaces including OpenSea and Magic Eden.
